# FinGuia — Agente Financeiro Inteligente com IA Generativa

Este projeto foi desenvolvido como parte de um desafio do bootcamp da **DIO** e tem como objetivo a criação de um **agente financeiro inteligente** utilizando **IA Generativa**.

O **FinGuia** é um assistente focado em **organização financeira para iniciantes**, atuando de forma educativa, consultiva e segura. O agente ajuda o usuário a entender seus gastos, acompanhar metas financeiras e aprender conceitos básicos de finanças pessoais, sempre utilizando dados contextuais e evitando recomendações financeiras indevidas.

O projeto demonstra conceitos como:
- Uso de **IA Generativa aplicada ao setor financeiro**
- Construção de agentes com **comportamento proativo**
- **Personalização de respostas** com base em contexto
- Estratégias de **segurança e anti-alucinação**
- Separação clara entre orientação educativa e aconselhamento financeiro

Todos os dados utilizados são **fictícios** e têm finalidade exclusivamente educacional.
